# Jerry的减压小锤 🎮

一个为女性玩家设计的轻度互动减压小游戏，基于《猫和老鼠》IP，玩家扮演Jerry通过点击Tom来释放压力。

## 🌟 游戏特色

- **无压力玩法**：没有失败惩罚，没有排名竞争
- **正向反馈**：每次点击都会给予温柔的文字鼓励
- **可爱画风**：柔和的粉色系配色，卡通风格的Tom角色
- **简单操作**：单指点击即可，随时随地可玩
- **治愈体验**：专注于减压和情绪释放

## 💻 技术栈

- **纯Web技术** - HTML5 + CSS3 + Vanilla JavaScript
- **Web Audio API** - 动态音效生成
- **CSS3动画** - 流畅的视觉效果
- **本地存储** - 设置持久化
- **响应式设计** - 支持各种设备屏幕

## 🚀 快速开始

### 方法1：使用Python启动服务器（推荐）

1. **确保已安装Python 3**
   ```bash
   python --version
   # 或者
   python3 --version
   ```

2. **启动游戏服务器**
   ```bash
   cd jerry-game
   python3 start-server.py
   ```

3. **打开浏览器**
   - 服务器会自动打开浏览器
   - 或手动访问：http://localhost:8000

### 方法2：直接在浏览器中打开

1. **下载所有文件到本地**
2. **双击 `index.html` 文件**
3. **游戏在浏览器中运行**

### 方法3：使用Live Server（VS Code）

1. **在VS Code中打开项目文件夹**
2. **安装Live Server扩展**
3. **右键点击 `index.html`**
4. **选择"Open with Live Server"**

## 🎮 游戏玩法

1. **点击Tom**：当Tom从网格中出现时，点击它进行"锤击"
2. **获得反馈**：每次成功击中都会显示鼓励文字和音效
3. **连击系统**：连续击中获得特殊奖励和反馈
4. **设置调整**：点击设置按钮控制音效、音乐和震动
5. **随时退出**：没有游戏结束，随时可以停止

## ✨ 核心功能

### 1. Tom角色系统
- **3x3网格**：Tom在9个位置随机出现
- **多种状态**：出现、待机、被击中、隐藏
- **流畅动画**：CSS3动画实现所有视觉效果
- **智能AI**：Tom有自己的行为模式

### 2. 正向反馈系统
- **即时反馈**：14种击中鼓励文字
- **连击奖励**：3连击、5连击、10连击特殊反馈
- **温柔支持**：15种暖心文字随机出现
- **视觉反馈**：星星特效、动画效果

### 3. 音效系统
- **动态生成**：使用Web Audio API实时生成音效
- **多种音效**：击中音、反馈音、UI音效
- **震动支持**：移动设备震动反馈
- **可调节音量**：支持静音模式

### 4. 设置管理
- **音效开关**：控制所有音效
- **音乐开关**：背景音乐控制
- **震动开关**：触觉反馈控制
- **自动保存**：设置本地持久化

## 🛠️ 自定义配置

### 修改游戏参数
在 `js/game.js` 中可以调整：
- Tom出现频率
- 连击重置时间
- 最大同时显示数量
- 反馈文字内容

### 添加新的反馈文字
在 `js/game.js` 的 `getInstantHitFeedback()` 和 `getGentleSupportText()` 方法中添加新文字。

### 自定义样式
在 `css/styles.css` 中修改：
- 颜色主题（CSS变量）
- 动画时长
- 布局尺寸
- 字体样式

## 🎯 设计理念

### 核心原则
- **减压优先**：不创造压力，只提供放松体验
- **温柔陪伴**：通过文字和画面给予情感支持
- **简单易用**：降低操作门槛，让任何人都能轻松上手
- **正向引导**：所有反馈都积极向上，鼓励玩家

### Tom人设
Tom是一个傻萌、笨笨、乐观的倒霉蛋。他不是敌人，而是陪玩家释放压力的小猫。被击中后只会懵、晕、卖萌，不会痛苦。

## 🔧 故障排除

### 游戏无法启动
1. **检查Python版本**：确保是Python 3.0+
2. **检查端口占用**：尝试使用不同端口 `python3 start-server.py 8001`
3. **浏览器兼容性**：使用现代浏览器（Chrome, Firefox, Safari）

### 音效不工作
1. **检查浏览器权限**：允许音频播放
2. **检查设置**：确保音效开关已开启
3. **刷新页面**：重新加载页面

### 触摸不响应
1. **检查设备**：确保支持触摸操作
2. **清除缓存**：刷新浏览器缓存
3. **重启游戏**：重新启动服务器
